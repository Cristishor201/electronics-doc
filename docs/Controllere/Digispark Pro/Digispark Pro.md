<h1>Digispark Pro</h1>

- board mini (mai mic decat Arduino PRO Mini)
- integrare USB pt flash
- poate folosi arduino IDE

- clock speed 16MHz
- AVR microcontroller
- 16KB Flash memorie (1.5KB bootloader)(mai putin decat Arduino UNO)
