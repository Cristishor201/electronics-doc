<h1>Arduino Programare - switch</h1>

```
switch(number){
  case 1:
    instructiune;
    break;
  case 2:
    instructiune;
    break;
  default: // default e obtional de pus ca cod
    instructiune;
}
```

sau:

```
switch(number){
  case 1: {
    instructiune1;
    instructiune2;
  }
    break;
  case 2:
    instructiune;
    break;
  default: //obtional
    instructiune;
}
```

Daca nu pui `break;` , programul va incerca sa caute daca se mai potriveste un caz mai jos.
Iar pentru ca nu vrem ca programul sa piarda timpul folosim `break;`

`default:` e obtional si se foloseste ca ultim block
